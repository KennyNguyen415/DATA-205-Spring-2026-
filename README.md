# DATA 205 Capstone Project: Maryland Insurance Appeals & Grievances

## Overview
This repository contains my capstone project for **DATA 205 at Montgomery College**. In collaboration with the **Maryland Insurance Administration (MIA)**, this project develops **interactive dashboards using Power BI** to analyze and compare trends and patterns from **2021 to 2025** related to insurance appeals and grievances.

- **Adverse Decision:** Occurs when an insurance carrier denies a claim because the service is deemed not medically necessary.
- **Grievance:** If a consumer disagrees with the adverse decision, they may file a grievance with the carrier, prompting an internal review of the service.
- **Complaint:** If the grievance is upheld and the consumer still believes the service should be covered, they can file a complaint with the **Maryland Insurance Administration (MIA)**, which conducts an independent review of the carrier’s decision.

Through data visualization and analysis, this project aims to provide insights into how these decisions evolve over time and across different insurance carriers.

## Data Sources
The data used in this project comes from the **public Appeals and Grievances Reports** available on the Maryland Insurance Administration website (maryland.gov). Each annual report contains eight appendices that provide detailed information on adverse decisions and grievance outcomes:

1. **Adverse Decisions by Carrier**
2. **Grievance Decisions by Carrier**
3. **Disposition of Carrier Grievance Decisions**
4. **Grievance Decisions by Carrier for Hospital Length of Stay (LOS)**
5. **Time Frame for Rendering a Grievance Decision by Carrier – Emergency Cases**
6. **Time Frame for Rendering a Grievance Decision by Carrier – Non-Emergency Cases**
7. **Internal Grievances Filed Considered Emergency Cases as Reported by Carrier**
8. **Administration Complaints**

## Tools & Technologies
- **Power BI** – Development of interactive dashboards and visualizations
- **Excel** – Initial data extraction and cleaning
- **GitHub** – Version control and project documentation

## Project Objectives
- Analyze trends in adverse decisions, grievances, and complaints from 2021–2025.
- Compare outcomes across insurance carriers.
- Identify patterns in emergency vs. non-emergency grievance cases.
- Provide clear, interactive visualizations to support data-driven insights.
